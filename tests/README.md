# Tests - AN Store

Thư mục này chứa tài liệu kiểm thử chính thức cho đồ án AN Store.

## File giữ lại

- `test-cases.md`: bộ test case chính thức theo module.
- `bug-list.md`: defect log tổng hợp, chỉ giữ nhóm lỗi quan trọng.

## File không nên giữ trong bản nộp

Các file checklist/report theo từng vòng fix bug là nhật ký làm việc nội bộ, không phải test case chính thức. Nội dung quan trọng đã được gom vào `test-cases.md` và `bug-list.md`.
